＜HTML＞
＜CSS＞
  • セミコロン ";" の付け忘れ注意!! それでも動いてしまうので気づきにくい、レイアウトが崩れた時にチェックしてみる。
  • "absolute" で "right:0" に指定したとき右に寄らず左に寄る場合は要素が画面左右全体に広がっていると思われる。
  • inline"であるのにも関わらず、要素が画面左右全体に広がっている場合は"width:100%"がになっていると思われる。
＜javaScript＞
＜jQuery＞
  • "id"の"#" や "class"の"." の付け忘れ注意!!
  • "id","class"呼び出し時、スペルミス注意、コピペしたほうがいい。
  • "id"名、"class"名、"data-"属性名　等に大文字を使うと、うまくいかない。使えない？
  • 要素が非表示("display:none"とか)になってると、座標が取得できない、その他の値もダメかも、、※未検証
  • "$(this)"を使って子要素を取得するときは、本当にその子要素を持っているか、よく確認する。　クリックイベントで自身の要素外を操作する時は、注意が必要。
  • $('this') ←誤　　　　$(this) ←正 ※" ' "いらない、たまにやる。

CSS

行間(縦)の空白

line-height: 24px;

文字サイズも含めた縦幅を設定する。
親要素の"height"と同じ値にすると、縦の中央揃えに使える。

文字間の空白

letter-spacing: 1px;

なし

boxの角丸め

border-radius: 10px;

border-radius: 左上 右上 右下 左下;

選択状態、不可

user-select: none;


ドラッグやダブルクリックでテキストを選択できない。
選択状態にならないので、色が変わらない

自動改行禁止

"width"指定


"width"を"px"指定すれば、それ以上縮まないのでウインドウの伸縮に影響されない。

jQuery

子要素全て取得

$('.element').find.('.ko').hide();

上記の場合全て'.ko'の要素が消える　　

直下の子要素のみ取得

$('.element').children.('.ko').hide();

上記の場合直下の'.ko'要素が消える。
　 ※↑間違い　要検証。　

属性の取得

$('element').attr('element');


＜属性の付加＞

$('ele').attr('id','name');

　　　　　　'ele'に'id'属性を追加して,属性値に'name'を与えている。
　　　　　　※元から'id'をもっている場合は属性値の変更になる。

要素の座標を取得する

$('.element').offset().top


$('.element').offset().top;　　　//"top"取得
$('.element').offset().left;　　//"left"取得

要素の位置にスクロール

$('html,body').scrollTop('top');


※"$('html,body')"は基本固定でつかう

　$('html,body').scrollTop(300);
　　　※scrollTop("飛びたい座標");

　$('html,body').animate({'scrollTop':300},500);
　　　※animate({'scrollTop':"座標"},"移動速度(ミリ秒)");

positon:の座標

$('.element').position();


$('.element').position().top;　　//"top"取得
$('.element').position().left;　//"left"取得

　　　※"right" "bottom" はないっぽい　※未検証
